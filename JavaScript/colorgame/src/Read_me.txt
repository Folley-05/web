--------------- COLORGAME -----------------------

HORLOGE est un jeux pour le fun qui permety de tester les capacites d'un developpeur a reconnaitre un code couleur rgb

le principe est le suivant :
un code couleur est donne et il faut trouver la couleur a laquelle il correspond parmi les 6 couleur qui sont propose

l'execution du programme necessite l'utilisation d'un serveur

Etat du projet : en cours de developpement
